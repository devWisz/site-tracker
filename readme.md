# Site Tracker

Site Tracker is a lightweight browser extension that helps you track the websites you visit most often.  
It allows you to save websites, monitor visit counts, and quickly access frequently used sites directly from your browser popup.

The extension focuses on simplicity and speed, making it useful for productivity and daily browsing habits.

## Features

Track Total Visits  
Keeps a count of how many times saved websites have been visited.

Most Visited Site  
Automatically identifies and displays the site you visit the most.

Save Current Tab  
Instantly save the website you are currently browsing.

Manual Site Entry  
Add any website manually to your saved list.

Visit Tracking  
Counts a visit when you stay on a saved website for a few seconds.

Saved History  
View all saved websites with their visit counts in one place.

Quick Open  
Open any saved website directly from the extension.

Delete Sites  
Remove specific sites from your saved list whenever you want.

Reset Data  
Clear all stored history and visit data instantly.

Lightweight and Fast  
Runs directly inside the extension popup without affecting browser performance.



## Use Cases

Productivity Tracking  
Understand which websites consume most of your time.

Quick Access Dashboard  
Keep frequently used tools, docs, or dashboards one click away.

Learning and Research  
Track which documentation or learning sites you visit most.

Focus Awareness  
Identify browsing habits and improve how you spend time online.

### Install Manually (Chrome / Chromium)

1. Download the latest release from the link above.
2. Extract the ZIP file.
3. Open your browser and go to:

chrome://extensions

4. Enable Developer Mode (top right).
5. Click Load unpacked.
6. Select the extracted extension folder.

The Site Tracker extension will now appear in your browser toolbar.

## How to Use

1. Click the Site Tracker extension icon.
2. Click Save Current Tab to store the website you are currently on.
3. Add websites manually using the input field if needed.
4. Click Show History to view saved websites and visit counts.
5. Open any saved site directly from the extension.
6. Reset all stored data anytime using Reset All.
